Executive Summary

มองเห็นความเหลื่อมล้ำ
ที่ซ่อนอยู่หลัง “ค่าเฉลี่ย”

GeoEquity แปลงภาพถ่ายดาวเทียมและข้อมูลโครงสร้างพื้นฐานให้เป็น Micro-level Wealth Index เพื่อช่วยภาครัฐชี้เป้าพื้นที่เปราะบางได้ละเอียดถึงระดับ 1 ตารางกิโลเมตร

ระบบผสานข้อมูลจาก THEOS-2, VIIRS, Sentinel-2 และ OpenStreetMap ผ่าน Deep Learning และ Machine Learning ก่อนสรุปเป็น Policy Dashboard สำหรับ Precision Budgeting การวางผังเมือง และการติดตามผลเชิงหลักฐาน

“จากข้อมูลพิกเซล สู่การตัดสินใจเชิงนโยบายที่ลงมือทำได้จริง”

Problem Definition

3 ช่องว่างของระบบประเมินความยากจนในปัจจุบัน

ปัญหาไม่ได้อยู่ที่ “ไม่มีข้อมูล” แต่อยู่ที่ข้อมูลหยาบ ล่าช้า และไม่ครอบคลุมประชากรที่เคลื่อนย้าย

ความย้อนแย้งของค่าเฉลี่ย

ตัวเลข GPP ระดับจังหวัดอาจบดบังชุมชนยากจนที่ซ่อนอยู่ในจังหวัดเศรษฐกิจขนาดใหญ่ ทำให้การจัดสรรงบไม่ตรงจุด

ความล่าช้าของข้อมูล

การสำรวจทีละครัวเรือนใช้เวลาและทรัพยากรสูง ข้อมูลจึงอาจล้าหลังจนไม่ทันต่อการแก้ปัญหาเชิงรุก

ประชากรแฝงตกหล่น

ผู้ย้ายถิ่นและชุมชนไม่เป็นทางการอาจไม่มีชื่อในทะเบียนราษฎร์ของพื้นที่ จึงไม่ถูกมองเห็นในระบบสวัสดิการ

Policy Implementation

3 เส้นทางนำไปใช้จริง

GeoEquity วางตำแหน่งเป็น “ปลั๊กอินจากอวกาศ” ที่เสริมระบบข้อมูลภาครัฐเดิม ไม่ใช่แทนที่ระบบที่มีอยู่

01

Precision Budgeting

ใช้ดัชนีความต้องการระดับไมโครประกอบการอนุมัติงบลงทุน ลดการพึ่งพาสัดส่วนประชากรหรือค่าเฉลี่ยระดับจังหวัดเพียงอย่างเดียว

02

Proactive Planning

จำลองจุดก่อสร้างถนน โรงเรียน หรือสถานีอนามัย เพื่อกระจายโครงสร้างพื้นฐานไปยังพื้นที่ที่ตกหล่นอย่างมีหลักฐานรองรับ

03

Evidence-based Evaluation

ติดตามภาพถ่ายดาวเทียมและดัชนีเศรษฐกิจก่อน–หลังดำเนินโครงการ เพื่อวัดผลสัมฤทธิ์ของเม็ดเงินลงทุนอย่างเป็นระบบ
